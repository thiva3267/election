How Duplicate Prevention Works
✅ No voter registry required
✅ Any email can vote — once
✅ On vote submission, email is checked against the Votes sheet
✅ If the email already exists → vote rejected instantly
✅ Race condition guard — double-checked before writing
✅ Admin can see every email that voted in the Vote Log tab
Setup Instructions
  1. Create a Google Sheet, copy its ID from the URL
  2. Go to script.google.com → New Project
  3. Paste the Code.gs file, set your SPREADSHEET_ID
  4. Run setupSheets() once — creates Candidates + Votes tabs
  5. Deploy → Web App (Execute as: Me, Access: Anyone)
  6. Paste the URL above and click Save & Test
